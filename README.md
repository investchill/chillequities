**ChillEquities** est le sibling de la galaxie [InvestChill](https://github.com/investchill)
dédié à la classe **equities globales développés** (developed-markets equities). Il s'inscrit
dans le folio All-Weather [ChillFOLIO](https://github.com/investchill/chillfolio) en couvrant
le quadrant macro **croissance haute / inflation basse**.

> ⚠️ **Phase amorçage.** Le projet est tout neuf. Le signal mensuel mécanique, le choix
> d'ETF concret et le backtest sibling-spécifique restent à figer. Cette page est une
> coquille en attendant.

## Cadre

- **Quadrant macro adressé** : croissance haute, inflation basse.
- **Cadre fiscal France** : préférence forte PEA via ETF UCITS synthétiques (World, S&P 500,
  MSCI Developed). CTO en repli si la PEA est saturée.
- **Mécanique attendue** : signal mensuel ON/OFF (cf. ChillBTC pour le template),
  anti-overfit (1-2 paramètres maximum, plateau de stabilité, walk-forward).
